개인정보 처리방침

최종 개정일: 2026년 6월 27일 · 시행일: 2026년 7월 1일

돌담(이하 "서비스")은 정보통신망 이용촉진 및 정보보호 등에 관한 법률, 개인정보 보호법 등 관련 법령을 준수합니다. 본 처리방침은 이용자의 개인정보가 어떻게 수집·이용·보관·파기되는지 안내합니다.

1-2. 수집하지 않는 항목

다음 정보는 일체 수집·저장하지 않습니다:

  • 휴대폰 번호 원본 (해시값만 저장)
  • 이름, 주민등록번호 (혼인관계증명서·청첩장에서 검토용으로만 사용 후 즉시 삭제)
  • 정확한 위치 정보 (GPS) — 시·도 수준만 수집
  • 통화 기록·문자·연락처
  • 카메라·갤러리 권한은 증명서·프로필 사진 업로드 시에만 요청, 자동 접근 없음

1-3. 수집 방법

  • 회원가입 단계에서 이용자가 직접 입력 또는 업로드
  • 서비스 이용 과정에서 자동 기록 (글 작성, 채팅, 신고 등)
  • 푸시 알림 발송을 위해 Expo Notifications SDK가 자동 발급한 토큰

회원 탈퇴 시 처리

회원이 직접 탈퇴를 요청하면 다음과 같이 처리됩니다:

  • 즉시 삭제: 휴대폰 번호 해시, 자기소개·관심사·직업·자녀 정보, 푸시 알림 토큰, 알림 내역, 차단 이력, 프로필 열람 이력
  • 익명 보존: 작성 글·댓글·투표·채팅 메시지는 다른 사용자의 대화 맥락 보존을 위해 유지하되, 작성자명은 "(탈퇴한 회원)"으로 표시되며 본인과 연결되지 않습니다
  • 회계상 보존: 포인트·결제 거래 내역은 관계 법령(전자상거래법, 국세기본법)에 따라 5년간 보관

4. 개인정보 제3자 제공

서비스는 원칙적으로 이용자의 개인정보를 외부에 제공하지 않습니다.

다만 다음의 경우 예외로 합니다:

수령자제공 항목제공 목적
법령에 따른 수사기관활동 기록, 신고 이력법원의 영장 또는 법령에 근거한 적법한 요청 시

6. 이용자의 권리와 행사 방법

이용자는 언제든지 다음 권리를 행사할 수 있습니다:

  • 열람: 마이페이지 → 프로필 편집에서 자신의 정보 조회
  • 정정·삭제: 닉네임 변경(월 3회), 자기소개·관심사·직업·자녀 정보 수정 또는 삭제
  • 처리 정지·동의 철회: 마이페이지 → 알림 설정에서 푸시 수신 중단
  • 차단 관리: 마이페이지 → 차단한 사용자 메뉴에서 차단·해제
  • 탈퇴: 마이페이지 → 탈퇴하기 메뉴에서 직접 탈퇴 (즉시 처리)

위 모든 권리는 앱 내에서 즉시 처리되며 별도 신청 절차가 없습니다. 권리 행사가 어려운 경우 아래 문의처로 연락해 주십시오.

7. 개인정보 파기 절차 및 방법

  • 파기 절차: 보유 기간 경과 또는 이용자 요청 시 즉시 파기
  • 파기 방법:
    • 전자적 파일: 복구 불가능한 방법으로 영구 삭제 (Cloudflare D1/R2의 삭제 API 호출)
    • 출력물: 파쇄 또는 소각 (해당 사항 없음 — 서비스는 종이 출력물을 보유하지 않음)

8. 개인정보의 안전성 확보 조치

서비스는 다음과 같은 기술적·관리적 조치를 시행합니다:

  • 휴대폰 번호 해시 처리: 원본 번호는 어디에도 저장되지 않으며 SHA-256 해시값만 사용
  • 증빙 서류 즉시 삭제: 혼인관계증명서·청첩장 모두 검토 완료 직후 원본 파일을 저장소에서 영구 삭제
  • 전송 구간 암호화: 모든 통신은 HTTPS/TLS로 암호화
  • 저장 구간 암호화: Cloudflare D1·R2의 기본 암호화 적용
  • 접근 통제: 관리자 페이지는 별도 토큰 인증 후 접근 가능, 접근 기록 보관
  • 세션 토큰 유효기간: 30일 자동 만료
  • 레이트리밋: 인증번호 발송 60초 간격, 시간당 5회 제한
  • 콘텐츠 필터링: 키워드 기반 개인정보 노출 방지 (전화번호·주민번호 패턴)

9. 아동의 개인정보 보호

본 서비스는 만 19세 이상의 이혼 경험이 있는 성인만 가입할 수 있습니다. 만 18세 미만 청소년은 가입할 수 없으며, 가입 시 제출되는 혼인관계증명서(상세)와 청첩장(해당 시)을 통해 자격을 확인합니다. 미성년자임이 확인되면 즉시 가입을 거부하며 관련 정보를 파기합니다.

10. 자동 수집 도구의 설치·운영 및 거부

서비스는 다음 자동 수집 도구를 사용합니다:

  • Expo Push Token: 푸시 알림 발송용. 앱 설정 → 알림 권한 해제로 수집 거부 가능
  • 에러 로그: 앱 충돌 시 익명 진단 정보. 개인 식별 정보 포함하지 않음

쿠키 등 웹 트래커는 사용하지 않습니다.

12. 처리방침의 변경
