안전

안전한 공간을 위한 약속

익명이라도 안전하게. 운영팀이 매일 살펴봅니다.

신고·차단

부적절한 회원·콘텐츠는 즉시 신고할 수 있고, 양방향 차단으로 서로 보이지 않게 됩니다.

자동 필터링

전화번호·주민번호 같은 개인정보 패턴은 자동으로 가려집니다.

법령 준수

개인정보보호법·정보통신망법 완전 준수. 휴대폰 번호는 SHA-256 해시로만 저장됩니다.

운영팀 모니터링

신고 5회 누적은 자동 숨김. 경고·일시정지·영구정지 단계별 제재로 어뷰징을 차단합니다.

자주 묻는 질문

가입 전, 궁금한 것들

가입 조건이 어떻게 되나요?
만 19세 이상의 대한민국 성인으로, 본인 명의 휴대폰과 이혼 사실이 기재된 혼인관계증명서(상세)를 제출할 수 있어야 합니다.
혼인관계증명서는 어떻게 처리되나요?
가입 시 한 번 업로드하시면 담당자가 검토 후 즉시 서버에서 영구 삭제합니다. 검토가 지연될 경우에도 최대 7일 후 자동 파기됩니다. 이름·주민번호는 어디에도 저장하지 않습니다.
다른 사용자가 제 이름이나 전화번호를 볼 수 있나요?
절대 보이지 않습니다. 자동 생성된 닉네임만 표시되며, 휴대폰 번호는 SHA-256 해시값으로만 저장되어 원본은 어디에도 남지 않습니다.
이용은 무료인가요?
게시판·투표·채팅 등 핵심 기능은 모두 무료입니다. 향후 다른 회원의 일부 프로필 항목 열람이나 폭파된 채팅방 부활 같은 부가 기능에 한해 포인트(유료 충전 가능)가 사용됩니다.
탈퇴는 자유롭게 할 수 있나요?
앱 내 마이페이지 → 탈퇴하기 메뉴에서 언제든 즉시 처리됩니다. 개인정보는 즉시 삭제되고, 작성하신 글·댓글은 다른 사용자의 맥락 보존을 위해 익명("(탈퇴한 회원)")으로 보존됩니다.
채팅방은 어떻게 만들어지나요?
직접 만들 수는 없고, "매칭 신청"을 누르면 비슷한 연령·성별·지역 조건의 4명(혼성 2M+2F)이 모이는 즉시 자동으로 방이 열립니다. 방은 3일 후 자동 종료되고, 멤버 과반이 "유지"를 선택하면 3일 연장됩니다.
출시는 언제인가요?
현재 베타 준비 중입니다. 정식 출시 알림을 받고 싶으시면 페이지 상단의 이메일 신청을 이용해 주세요. 베타 참여 문의는 support@doldam.app으로 보내주시면 됩니다.
