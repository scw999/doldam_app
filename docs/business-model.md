# 돌담(Doldam) — 비즈니스 모델

> Claude Code/인수인계자를 위한 수익 구조·기능 티어·가격 가이드.
> 구현 변경 시 이 문서도 함께 갱신.

---

## 1. 한 줄 요약

**인증된 돌싱 전용 커뮤니티**에서 *감정 공감 루프*는 무료로 주고,
*자기 정보·타인 반응 확인·방 수명 연장* 같은 **"호기심·관계 욕구"** 액션만 유료화.

데이팅앱처럼 "매칭을 팔지" 않는다 — 돌담은 **돌싱이 자기 회복 기간에 머무는 공간**이 목표.

### C. 포인트 무상 지급 루프 (LTV 유지)

사용자를 붙잡아 두는 "매일 돌아올 이유".

| 행동 | 지급 | 캡 |
|------|------|-----|
| 일일 로그인 | +3P | 1일 1회 |
| 기분 기록 | +3P | 1일 1회 |
| Q&A 미션 (3일/10답) | 최대 +100P | 3일당 1회 |

**30일 FIFO 만료** → 오래 안 쓰면 자연 소멸, 구매/활동 동기 유지.

## 3. 기능 티어 분리 (무료 vs 유료)

### 무료 (완전히 오픈)
- 휴대폰·증명서 인증
- 게시판 읽기/쓰기/반응
- 찬반·선택형 투표 참여·생성
- 소그룹 매칭·3일 채팅
- 감정 타임라인 (본인만)
- 프로필 **요약** (나이대/성별/지역 등)
- 채팅방 "서로의 느낌" 투표 참여

### 유료 (포인트 또는 IAP)
- **소그룹 방 부활** — 3일 더
- **프로필 세부 항목** — 직업군/관심사/자녀 유무 등 개별 잠금 해제
- **"나를 뽑은 사람 보기"** — 채팅방 TBH 투표에서 나를 선택한 사람 닉네임 공개
- (미래) 메시지 읽음 확인, 닉네임 재롤, 프리미엄 테마방 개설

### 정책 가드 (유료 모델 보호)
- **피어 폴은 응답이 있으면 삭제 불가** — "나를 뽑은 사람 보기" 과금 대상 데이터를 작성자가 임의로 지울 수 없게.
  0 응답 상태에선 삭제 가능(오타 대응).
- **채팅 메시지는 vote 삭제와 무관하게 유지** — 채팅 로그 무결성 + 과금 대상 데이터 보존.
- **인증서 R2는 OCR 후 즉시 삭제** — 이건 수익과 반대 방향의 *법적 안전장치*.

## 4. 경쟁 포지셔닝

| 축 | 데이팅앱(Tinder, 아만다 등) | 커뮤니티(블라인드 등) | **돌담** |
|----|---|---|---|
| 목적 | 매칭·만남 | 익명 하소연 | **감정 회복 + 약한 연결** |
| 인증 | 사진·직장 | 회사 이메일 | 휴대폰 + 혼인관계증명서 |
| 매칭 방식 | 1:1 스와이프 | X | 소그룹 6~8명, 3일 만기 |
| 유료화 포인트 | "매칭 더 보기" | 광고 | **자기 공개·상대 반응** |
| 여성/미성년 보호 | 약함 | — | **증명서 필수, 성별 전용 공간** |

차별화: *"진짜 돌싱끼리만의 익명 공간"* + *느슨한 호기심 과금*.

## 5. KPI (운영 지표 후보)

- **D7/D30 리텐션** — 기분 기록·Q&A 미션 참여율이 선행 지표
- **포인트 소진율** — 무상 지급 vs 과금 비율
- **방 부활 전환율** — 만료된 방 중 부활된 비율 (과금 트리거)
- **피어 폴 응답률** — 채팅방 내 사용자 참여도
- **"나를 뽑은 사람 보기" 전환율** — 피어 폴의 유료 공개 클릭률 ← 가장 핵심 수익 KPI

## 6. 준법·윤리 리스크

- **정보통신망법**: 개인정보 수집 최소화, 이혼 여부를 넘어선 민감정보 저장 금지
- **통신비밀보호법**: 채팅 로그 보관 기간 명시 (3일 방 수명 + 만료 후 N일 후 완전 삭제)
- **청소년보호법**: 성인 인증 필수
- **프리미엄 기능의 과도한 푸시 금지**: "결제 유도" 알림은 하루 1회 이내
- **"나를 뽑은 사람 보기"는 익명성 일부 훼손**  
  → 공개 범위를 "참여자 수만" → "포인트로 닉네임까지" 단계적으로 설계 필요
  → 약관에 고지 + 회원 가입 시 동의 필수

## 7. 로드맵과 연결

- **Phase 3**: 포인트 경제 (무상 지급 + 방 부활 기초) ← 현재
- **Phase 4**: 채팅 기반 피어 폴 (방어 지표: 참여율)
- **Phase 5**: **"나를 뽑은 사람 보기" 유료화** + 프로필 세부 공개
- **Phase 6**: IAP 구매, A/B 가격 테스트
- **Phase 7**: 광고 없음 강조 마케팅
