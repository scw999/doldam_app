# Pretendard 폰트 설치

Pretendard는 Google Fonts에 없으므로 로컬 .otf 파일을 직접 추가해야 합니다.

## 다운로드

https://github.com/orioncactus/pretendard/releases 최신 릴리즈에서 `Pretendard-*.zip` 다운로드.

## 이 디렉토리에 필요한 파일

- `Pretendard-Regular.otf` (400)
- `Pretendard-Medium.otf` (500)
- `Pretendard-SemiBold.otf` (600)
- `Pretendard-Bold.otf` (700)

릴리즈 zip 안 `public/static/` 경로에 있습니다. 위 4개만 이 폴더로 복사하세요.
